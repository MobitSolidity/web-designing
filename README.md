# هدیه‌یار – نسخه استاتیک آزمایشی

این مخزن اکنون شامل پیاده‌سازی جدید فروشگاه کادو در فولدر `giftshop-endweb` است (منتقل‌شده از `endweb-1.2v`). نسخه قبلی (قالب `cluade-make`) کنار گذاشته شده و در توسعه آینده استفاده نمی‌شود. صفحات اصلی شامل خانه، فهرست محصولات، صفحه محصول و سبد خرید هستند و همگی در ساختار استاتیک موبایل‌اول پیاده‌سازی شده‌اند.
